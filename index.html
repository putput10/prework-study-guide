HTML

  • The head element contains the information about the webpage.
  • The body element represents the visible content shown to the user.
  • The id attribute identifies elements and selects them in an HTML document.
  • The style element the CSS styles associated with a webpage.
  • The base element defines the base URL for a webpage.
  • The link element connects an external resource to the HTML document.
  • The meta element defines the information in a webpage.
  • The img element holds information about images on the webpage.
  • The src attribute specifies the location of the image grom the img element.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Declarations contain the CSS property we want to apply and the value of the property.
  • The * symbol declares that all the elements on the page will have the same style properties.

Git

  • git staus: checks what branch we are currently on.
  • git checkout -b branch name: creates a new branch name and switches to it.
  • git push: Moves local branch commits to the remote repository branch.
  • git pull: fecthes and merges all commits from the tracking remote branch.
  • git add {file}: adds a file as is to you next commit (stage).
  • git config --global user.name "{firstname lastname}: sets an identifiable name for other users and for credit.
  • git config --glibal user.email "{email}": sets an email address for history markers.
  • git clone {URL}: Acquire an entire repositroy from a hosted location using URL.
  • git log: Shows all commits in the current branch's history.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Console refers to a test environment that developers use to check out their code.
  • .log() is a set of instructions that can be executed by a computer.
  • "if" statements are ways of introducing decsion-making in your code.
  • "else if" allows for more than one condition to be usede in JavaScript.
  • The SEO (===) checks to see if two values are equal and results in either true or false.
  • An array is a single variable that is used to hold a group of data.
  • A function is a set of instructions that tells the computer how to perform a certain task.
